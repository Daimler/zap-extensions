Сохранить выбранные записи как HAR (файл архива HTTP)

Пункт контекстного меню для сохранения выбранных HTTP-сообщений в формате HAR.

Сохранить необработанное сообщение

Предоставляет контекстное меню для сохранения содержимого сообщений HTTP в двоичном виде.

Сохранить XML-сообщение

Предоставляет контекстное меню для сохранения содержимого сообщений HTTP в формате XML.

Показывает контекстное меню для сохранения сообщений HTTP в формате XML.

Возможность импорта сообщений из HTTP-архива (HAR), доступная через меню «Импорт».

Импорт файла журнала

Позволяет импортировать файлы журналов из ModSecurity и файлы, ранее экспортированные из ZAP.

Импорт URL-адресов

Возможность импорта файла URL-адресов доступна через меню «Импорт» («Импорт файла, содержащего URL-адреса»). Файл должен быть обычным текстом с одним URL-адресом в строке. Пустые строки и строки, начинающиеся с #, будут игнорироваться.

It also supports the Automation Framework.

ZAP API

Это дополнение также предоставляет различные конечные точки API ZAP для облегчения программного использования этой функции.
  • /exim/importHar (filePath*)
  • /exim/importModsec2Logs (filePath*)
  • /exim/importUrls (filePath*)
  • /exim/importZapLogs (filePath*)
